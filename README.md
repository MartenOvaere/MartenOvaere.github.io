# MartenOvaere.github.io (Quarto)

This repository hosts the source for your academic website, built with **Quarto** and deployed to **GitHub Pages** via **GitHub Actions**.

## Local preview (optional)

1. Install Quarto: https://quarto.org/docs/get-started/
2. From the repo root:

```bash
quarto preview
```

## Deploy

Push to `main`. GitHub Actions will build and publish automatically.

## Updating your photo

Replace `assets/profile.svg` with a real photo named `assets/profile.jpg` and update `index.qmd` to point to it.

## Updating your CV link

Upload your CV as `assets/Marten_Ovaere_CV.pdf` and update `cv.qmd` (recommended).
